Insert Image file in top left
Hit Import

The image will appear in bottom left. 

Select a spot on the image. That color will appear in the box above color wheel.
Select a spot on the color wheel. That color will appear in the box below color wheel.
The difference between the two will be stored as a tuple in the bottom right.
The color that is the result of the applying the difference to the image selection will be shown in the box right of the wheel.
Selecting  a new spot on the image keeps the difference and reapplies the operation
Selecting a new spot on the wheel recalculates teh difference, resetting the process.
